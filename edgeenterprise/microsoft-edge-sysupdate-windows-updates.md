## <span data-ttu-id="7e248-107">Microsoft Edge und der Freigabezyklus der Windows-Versionen</span><span class="sxs-lookup"><span data-stu-id="7e248-107">Microsoft Edge and the Windows release cycle</span></span>

<span data-ttu-id="7e248-108">Die nächste Version von Microsoft Edge bietet häufigere und flexiblere Aktualisierungsfunktionen.</span><span class="sxs-lookup"><span data-stu-id="7e248-108">The next version of Microsoft Edge features more frequent and more flexible updating capabilities.</span></span> <span data-ttu-id="7e248-109">Da Browserversionen nicht an die Hauptversionen von Windows gebunden sind, werden Änderungen am Betriebssystem vorgenommen, um sicherzustellen, dass die nächste Version von Microsoft Edge nahtlos in Windows passt.</span><span class="sxs-lookup"><span data-stu-id="7e248-109">Because browser releases aren't bound to the Windows major releases, changes will be made to the operating system to ensure that the next version of Microsoft Edge fits seamlessly into Windows.</span></span> <span data-ttu-id="7e248-110">Aus diesem Grund werden Featureupdates in einem (etwa) 6-wöchigen Zyklus veröffentlicht.</span><span class="sxs-lookup"><span data-stu-id="7e248-110">As a result, feature updates will be released on a 6-week cycle (approximately).</span></span> <span data-ttu-id="7e248-111">Sicherheits- und Kompatibilitätsupdates werden nach Bedarf ausgeliefert.</span><span class="sxs-lookup"><span data-stu-id="7e248-111">Security and compatibility updates will be shipped as needed.</span></span>

## <span data-ttu-id="7e248-112">Updates und die Benutzererfahrung</span><span class="sxs-lookup"><span data-stu-id="7e248-112">Updates and the user experience</span></span>

<span data-ttu-id="7e248-113">Updates ändern die Benutzererfahrung erst, wenn der Stable-Kanal der nächsten Version von Microsoft Edge installiert ist.</span><span class="sxs-lookup"><span data-stu-id="7e248-113">Updates won’t change the user experience until the Stable channel of the next version of Microsoft Edge is installed.</span></span> <span data-ttu-id="7e248-114">Die Installation von Microsoft Edge Beta, Dev oder Canary löst keine Änderungen in Windows aus.</span><span class="sxs-lookup"><span data-stu-id="7e248-114">Installing Microsoft Edge Beta, Dev, or Canary won’t trigger any changes in Windows.</span></span> <span data-ttu-id="7e248-115">Diese Browserversionen werden zusammen mit vorhandenen Browsern installiert.</span><span class="sxs-lookup"><span data-stu-id="7e248-115">These browser releases will be installed alongside existing browsers.</span></span>

<span data-ttu-id="7e248-116">Wenn alle Updates angewendet wurden UND der Stable-Kanal der nächsten Version von Microsoft Edge auf Systemebene installiert ist, werden die folgenden Änderungen auf dem System wirksam:</span><span class="sxs-lookup"><span data-stu-id="7e248-116">When all the updates are applied AND the Stable channel of the next version of Microsoft Edge is installed at the system-level, the following changes will take effect on the system:</span></span>

- <span data-ttu-id="7e248-117">Alle Startmenü-Pins, Kacheln und Verknüpfungen für die aktuelle Version von Microsoft Edge werden auf die nächste Version von Microsoft Edge migriert.</span><span class="sxs-lookup"><span data-stu-id="7e248-117">All start menu pins, tiles, and shortcuts for the current version of Microsoft Edge will migrate to the next version of Microsoft Edge.</span></span>
- <span data-ttu-id="7e248-118">Alle Taskleisten-Pins und Verknüpfungen für die aktuelle Version von Microsoft Edge werden auf die nächste Version von Microsoft Edge migriert.</span><span class="sxs-lookup"><span data-stu-id="7e248-118">All taskbar pins and shortcuts for the current version of Microsoft Edge will migrate to the next version of Microsoft Edge.</span></span>
- <span data-ttu-id="7e248-119">Die nächste Version von Microsoft Edge wird an die Taskleiste angeheftet.</span><span class="sxs-lookup"><span data-stu-id="7e248-119">The next version of Microsoft Edge will be pinned to the taskbar.</span></span> <span data-ttu-id="7e248-120">Wenn die aktuelle Version von Microsoft Edge bereits angeheftet ist, wird sie ersetzt.</span><span class="sxs-lookup"><span data-stu-id="7e248-120">If the current version of Microsoft Edge is already pinned, it will be replaced.</span></span>
- <span data-ttu-id="7e248-121">Die nächste Version von Microsoft Edge fügt eine Verknüpfung auf dem Desktop hinzu.</span><span class="sxs-lookup"><span data-stu-id="7e248-121">The next version of Microsoft Edge will add a shortcut to the desktop.</span></span> <span data-ttu-id="7e248-122">Wenn für die aktuelle Version von Microsoft Edge bereits eine Verknüpfung vorhanden ist, wird sie ersetzt.</span><span class="sxs-lookup"><span data-stu-id="7e248-122">If the current version of Microsoft Edge already has a shortcut, it will be replaced.</span></span>
- <span data-ttu-id="7e248-123">Die meisten Protokolle, die standardmäßig von Microsoft Edge behandelt werden, werden auf die nächste Version von Microsoft Edge migriert.</span><span class="sxs-lookup"><span data-stu-id="7e248-123">Most protocols that Microsoft Edge handles by default will be migrated to the next version of Microsoft Edge.</span></span>
- <span data-ttu-id="7e248-124">Die aktuelle Version von Microsoft Edge wird auf allen UX-Oberflächen im Betriebssystem ausgeblendet, einschließlich der Einstellungen, aller Apps und aller Dialogfelder zur Unterstützung von Dateien oder Protokollen.</span><span class="sxs-lookup"><span data-stu-id="7e248-124">Current Microsoft Edge will be hidden from all UX surfaces in the OS, including settings, all apps, and any file or protocol support dialogs.</span></span>
- <span data-ttu-id="7e248-125">Bei allen Versuchen, die aktuelle Version von Microsoft Edge zu starten, wird auf die nächste Version von Microsoft Edge umgeleitet.</span><span class="sxs-lookup"><span data-stu-id="7e248-125">All attempts to launch the current version of Microsoft Edge will redirect to the next version of Microsoft Edge.</span></span>

  > [!NOTE]
  > <span data-ttu-id="7e248-126">Im Falle einer Installation auf Benutzerebene werden diese Verhaltensweisen nicht ausgelöst.</span><span class="sxs-lookup"><span data-stu-id="7e248-126">User-level installs won’t trigger these behaviors.</span></span>

<span data-ttu-id="7e248-127">Zusammen mit den vorherigen Änderungen gibt es Änderungen, die unabhängig davon stattfinden, ob der Stable-Kanal der nächsten Version von Microsoft Edge installiert ist.</span><span class="sxs-lookup"><span data-stu-id="7e248-127">Along with the previous changes, there are changes that will happen regardless of whether the Stable channel of the next version of Microsoft Edge is installed.</span></span>

- <span data-ttu-id="7e248-128">Microsoft Edge hebt die Registrierung für Bücher und XML-Protokolle auf, die von der nächsten Version von Microsoft Edge nicht unterstützt werden.</span><span class="sxs-lookup"><span data-stu-id="7e248-128">Microsoft Edge will de-register for the books and XML protocols that the next version of Microsoft Edge doesn't support.</span></span> <span data-ttu-id="7e248-129">Benutzer, die versuchen, diese Protokolle zu öffnen, werden in einem Dialogfeld aufgefordert, eine Standard-App auszuwählen.</span><span class="sxs-lookup"><span data-stu-id="7e248-129">Users attempting to open these protocols will get a dialog that prompts them to choose a default app.</span></span> <span data-ttu-id="7e248-130">Weitere Informationen zu Änderungen an Büchern finden Sie unter [Herunterladen einer ePub-App zum Lesen von E-Books](https://nam06.safelinks.protection.outlook.com/?url=https%3A%2F%2Fsupport.microsoft.com%2Fhelp%2F4517840&data=02%7C01%7Cv-danwes%40microsoft.com%7Cc9f8571b880549c30fcf08d72be5eaf9%7C72f988bf86f141af91ab2d7cd011db47%7C1%7C0%7C637026138803983526&sdata=qtb3DvVZQ6H%2FFXnBievkl%2B%2BngAQXwl340PcH8kRc3y4%3D&reserved=0).</span><span class="sxs-lookup"><span data-stu-id="7e248-130">Learn more about changes to books support at [Download an ePub app to keep reading e-books](https://nam06.safelinks.protection.outlook.com/?url=https%3A%2F%2Fsupport.microsoft.com%2Fhelp%2F4517840&data=02%7C01%7Cv-danwes%40microsoft.com%7Cc9f8571b880549c30fcf08d72be5eaf9%7C72f988bf86f141af91ab2d7cd011db47%7C1%7C0%7C637026138803983526&sdata=qtb3DvVZQ6H%2FFXnBievkl%2B%2BngAQXwl340PcH8kRc3y4%3D&reserved=0).</span></span>
